# PDD_dev (Assets + Catalog)

## What’s inside

- `docs/products.json`: product catalog for the mini program
- `docs/images/**`: product images (organized by category)

## GitHub Pages

Enable GitHub Pages with:

- Settings -> Pages
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

After it’s enabled, `products.json` will be available at:

`https://<github-username>.github.io/PDD_dev/products.json`

Images will be available at:

`https://<github-username>.github.io/PDD_dev/images/<category>/<file>.jpg`
